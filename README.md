# MonitorCookie
一款用来监听全局cookie变化的Chrome插件



### 已实现功能

- 监听浏览器全局 cookie 改变
  - 指定关键词进行过滤
- 针对 cookie 变化通过 webhook 自动转发



### 安装方法

手动安装:

- 下载 MonitorCookie.zip 并解压
- 浏览器打开 `选项 -> 更多工具 -> 扩展程序`
- 开启 `开发者模式`
- 点击 `加载已解压的扩展程序`，选择第一步解压后的文件夹



### 使用

- 点击插件图标，打开配置页面
- 指定 webhook 地址，点击 修改 按钮
- 勾选 监听
